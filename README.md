# 🪷 Bhagavad Gita Wisdom Bot

An AI-powered spiritual guide that provides timeless wisdom from the Bhagavad Gita. This project uses a RAG (Retrieval-Augmented Generation) system to answer questions based on the sacred text, integrated with Supabase for secure authentication and history management.

![Bhagavad Gita Bot](frontend/images/icon-512.png)

## 🌟 Features
- **AI-Powered Insights**: Uses Groq (Llama 3) and LangChain for deep spiritual guidance.
* **Supabase Integration**: Managed authentication and persistent conversation history.
- **Verse of the Day**: Rotating daily wisdom on the landing page.
- **PWA Support**: Installable on mobile and desktop for instant access.
- **Premium UI**: Sleek, gold-accented design with a mindful aesthetic.

## 🚀 Tech Stack
- **Backend**: Flask, LangChain, Groq AI, FAISS (Vector DB).
- **Frontend**: Vanilla JS, Vanilla CSS, Supabase JS SDK.
- **Database/Auth**: Supabase.
- **Deployment**: Docker, Hugging Face Spaces.

## 🛠️ Setup Instructions

### 1. Prerequisites
- Python 3.11+
- Supabase Account
- Groq API Key

### 2. Supabase Configuration
Run the following SQL in your Supabase SQL Editor (available in `supabase_setup.sql`):
```sql
CREATE TABLE conversations (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  user_id UUID REFERENCES auth.users(id) ON DELETE CASCADE,
  question TEXT NOT NULL,
  answer TEXT NOT NULL,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

ALTER TABLE conversations ENABLE ROW LEVEL SECURITY;
-- (Add policies for SELECT, INSERT, DELETE based on auth.uid())
```

### 3. Backend Setup
Create a `backend/.env` file:
```env
GROQ_API_KEY="your_groq_key"
SUPABASE_URL="https://your-project.supabase.co"
SUPABASE_KEY="your-service-role-key"
JWT_SECRET_KEY="your-secret"
```

Install dependencies and run:
```bash
cd backend
pip install -r requirements.txt
python ingest.py  # Rebuilds the FAISS index
python app.py     # Starts the server
```

### 4. Frontend Setup
Update `frontend/auth.js` with your Supabase Anon keys for direct client-side integration.

## 🚢 Deployment (Hugging Face Spaces)
1. Push this repository to GitHub/Hugging Face.
2. Set the following **Secrets** in your HF Space:
   - `GROQ_API_KEY`
   - `SUPABASE_URL`
   - `SUPABASE_KEY`
   - `JWT_SECRET_KEY`
3. The `Dockerfile` handles the rest!
